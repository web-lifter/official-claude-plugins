# home-life-logistics

4 lifestyle skills for the planning bits of life.

| Skill | Purpose |
|---|---|
| `trip-day-by-day` | Multi-day travel itinerary with pacing rules, transfers, must-book windows |
| `home-tlc-calendar` | Annual home-maintenance calendar by home type (AU-specific compliance) |
| `adulting-checklist` | Quarterly life-admin sweep tailored to life stage |
| `thoughtful-gifts-plan` | Annual gift plan from relationship inventory, budget, lead-times |

Pure reasoning — no external APIs required.
