# personal-finance

5 lifestyle skills for Australian personal-finance planning.

| Skill | Purpose |
|---|---|
| `money-map` | Zero-based or 50/30/20 budget with pay-day automation routing |
| `debt-knockout-plan` | Avalanche vs snowball comparison, payoff timeline, refinance scan |
| `savings-game-plan` | Savings-rate target with sinking funds and automation triggers |
| `future-me-projection` | Retirement projection with AU super + sensitivity (FIRE scenarios) |
| `rainy-day-plan` | 6-month buffer sizing + insurance gap audit + emergency playbook |

**Disclaimer:** General information only. This is not personal financial advice as defined by Australian financial-services regulation (ASIC). Consult a licensed financial adviser before acting.
