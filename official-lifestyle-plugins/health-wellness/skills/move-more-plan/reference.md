# Move More Plan — Reference Material

## Exercise Library by Equipment Tier

### Bodyweight only

- Squat / split squat / Bulgarian split squat / pistol progression
- Push-up (incline → flat → decline → archer → one-arm progression)
- Pull-up (inverted row → assisted chin-up → chin-up → pull-up)
- Lunge (forward / reverse / walking)
- Single-leg glute bridge
- Hollow body / dead bug / side plank
- Burpees (low-impact: step-back)
- Step-ups onto a sturdy chair

### Dumbbells + bands

- Goblet squat
- DB Romanian deadlift
- DB bench press (floor or bench)
- DB single-arm row
- DB overhead press
- Walking lunge
- Banded pull-aparts; banded face-pulls

### Home gym (rack + barbell)

- Back squat / front squat
- Conventional deadlift / Romanian deadlift / sumo deadlift
- Bench press / overhead press
- Pendlay row / Yates row
- Chin-up / pull-up (bar in rack or doorway)
- Hip thrust

### Full gym (commercial)

- All home-gym options, plus:
- Hack squat / leg press / leg curl / leg extension
- Cable rows / lat pulldowns / cable flyes
- Pec deck / chest-supported rows
- Smith machine variations
- Specialty bars (trap bar, safety squat bar)

## Substitution Decision Table

| Want | Use this | Why |
|------|---------|-----|
| Knee bothers on back squat | Goblet squat / split squat | Less compressive; easier to bail |
| Lower back fatigued | RDL instead of deadlift | Less spinal load |
| No bench | Floor press / push-up variants | Same press pattern |
| No pull-up bar | Inverted row (table edge) | Same pull pattern, lower intensity |
| Short on time | Supersets (e.g. squat + row) | Cuts session by ~30% |
| Travel | Bodyweight + bands | Packs in a carry-on |

## Deload Triggers

| Signal | Threshold | Action |
|--------|----------|--------|
| Sleep < 6h | 3 nights in a row | Deload immediately |
| RPE for working sets | Climbing 1 point/wk | Plan deload next week |
| Same load stalled | 2 sessions in a row | Deload that lift only |
| Joints aching outside sessions | 2 days | Skip the bothered movement; sub in |
| Mood notably low | 5+ days | Cut volume −30%; recheck sleep + nutrition |
| Resting HR up | +5–10 bpm above baseline | Recovery issue — full deload week |

---

## Volume Heuristics (sets per muscle group per week)

- Maintenance: 4–6 sets/wk
- Hypertrophy: 10–20 sets/wk
- Above 20 sets/wk: only in late blocks for advanced lifters

Junk volume rule: if RIR > 4 on every set, the set was likely junk. Track quality, not just quantity.

---

## Periodisation Models

- **Linear** — load increases week-to-week; reset every 4–6 weeks. Best for novices.
- **DUP (daily undulating)** — each session varies (heavy / volume / power). Better for intermediates.
- **Block** — 4-week strength → 4-week hypertrophy → 4-week peak. Common for advanced.
- **Conjugate** — multiple qualities trained simultaneously. Powerlifting-focused; advanced only.
