## Program at a Glance

- **Goal:** Build a baseline of strength + cardiovascular fitness; ~12 months consistent training to date = 0
- **Length:** 12 weeks
- **Sessions per week:** 3 (full-body, every other day)
- **Equipment tier:** Bodyweight + a doorway pull-up bar ($30 from Bunnings)
- **Estimated session time:** 30–40 min including warm-up + cool-down

### Weeks 2–12 — linear progression

| Block | Weeks | Focus | Progression rule |
|-------|-------|-------|------------------|
| Foundation | 1–4 | Master form; add reps each week if all sets clean | Squat backpack +2kg/wk; push-up +1 rep/wk |
| Build | 5–8 | Add a second exercise per pattern; introduce timed intervals | Add 2nd set of harder variation (e.g. deficit push-up) |
| Consolidate | 9–11 | Increase volume; harder variations | Aim for 12 reps consistent on all top sets |
| Test + redesign | 12 | Test 1RM-equivalent (max-rep push-ups, max-rep chin-ups, max-rep bodyweight squats) | Redesign next block — likely time to invest in dumbbells |

## Progression Rules

- **Squat (backpack):** add 2kg/wk; if can't complete all sets clean, repeat same load next session.
- **Push-up:** when 3×10 strict, progress to a harder variation (decline → pike → deficit).
- **Inverted row:** when 3×12 strict, lower the angle (steeper = harder).
- **Cardio:** 3× weekly walks of 20–40 min; aim for a "talking but not singing" pace.

## Recovery Cues

- **Sleep:** target ≥ 7h. Skip the harder variation if sleep was < 6h.
- **Soreness:** day-after soreness is normal first 4 weeks. Sharp joint pain ≠ soreness — stop and book a physio.
- **Active recovery:** Tue / Thu / Sat / Sun walks — short, light, outside if possible.
- **Nutrition:** 1.6 g protein/kg bodyweight on training days; eat normally otherwise.
