# health-wellness

5 lifestyle skills for evidence-rated nutrition, training, sleep, and supplements.

| Skill | Purpose |
|---|---|
| `week-of-meals` | Weekly meal plan with macro targets, prep-day workflow, AUD shopping list, fallbacks |
| `move-more-plan` | N-week strength/endurance/hybrid training program with periodisation + substitutions |
| `sleep-tune-up` | Audit current sleep + prescribe 14-day routine, environment, light/caffeine timing |
| `smart-supplement-stack` | Evidence-rated supplement stack with timing, dose, cycling, interactions |
| `daily-wellness-stack` | Health-friendly habit stack (hydration / movement / sunlight / breathwork) |

**Disclaimer:** This plugin produces general information only — not medical, dietary, or fitness advice. Consult a qualified clinician before making changes.
