# Habit Stacker — Reference Material

## Habit-Stack Patterns Library

### Morning stack (after waking)
- Coffee → 10 squats → top-3 notebook → 5-min stretch
- Bathroom → cold splash → glass of water → vitamin
- Light on → bed made → window opened → journal entry

### Transition stack (between work modes)
- Close laptop → 2-min walk outside → water → next-block intention
- Last meeting → 90-second box-breathing → top-3 review
- Lunch break → screen off → leave the building → 10 deep breaths before re-entry

### Evening stack (wind-down)
- Kids' bath → book on pillow → kettle for tea → screens off at 9pm
- Dishes done → diffuser on → stretch → bedside book open
- Last work email closed → laptop in drawer → kettle → planning for tomorrow

### Recovery stack (post-illness, post-travel, post-collapse)
- Wake → 30-sec breathing → glass of water → one squat → done
- The point is to fire the cue. Volume is irrelevant. Two days of cue-firing is the entire goal.

- "I'm becoming someone who treats sleep as a non-negotiable, not a leftover."

## Friction-Design Checklist

### Reduce friction for desired habits
- Move the object you need to *first thing* you'll see (book on pillow, kettle visible, shoes by door)
- Pre-decide outfit, food, or first task the night before
- Set up the room for the habit the night before (yoga mat unrolled, glass beside bed)
- Default the app open / file open / page open
- Auto-pour, auto-fill, auto-route (standing orders, subscriptions, calendar blocks)

### Add friction for undesired habits
- Phone in another room while doing the desired habit
- Sign out of social apps each night; require re-login each time
- TV remote in a drawer; controller batteries removed
- Snack foods out of sight, on a high shelf, or not bought
- Add a 30-second delay between cue and undesired action (e.g. "before I scroll, do 5 push-ups")

### Environment redesign — full list
| Reduce friction | Add friction |
|----------------|-------------|
| Book on pillow | Phone in another room |
| Pre-filled water bottle on desk | Snack jar moved to garage |
| Kettle near workout space | TV remote in a drawer |
| Workout clothes laid out night before | Notifications off by default |
| Pre-planned meal in the fridge | Sign-out of social each night |
| App pinned to home screen | App in a folder named "Reconsider" |
| Default to first task open at desk | Auto-deleted draft folder |
| Calendar block for the habit | 10-minute walk before opening laptop |
| Habit anchored to an existing cue | Phone in greyscale mode |
| Tracker visible from where you'll be | Wallet locked in a drawer |

## Tiny Habits Scoring Matrix

Score each candidate habit on motivation (1–5) and ability (1–5). The recommended prescription:

| Motivation \ Ability | Low ability (1–2) | Medium ability (3) | High ability (4–5) |
|---------------------|-------------------|-------------------|-------------------|
| **Low motivation (1–2)** | Drop — wrong habit, wrong time | Make it smaller, add reward, increase visibility | Likely fine — just needs cue |
| **Medium motivation (3)** | Shrink to under 30 seconds | Standard habit-stack candidate | Schedule and forget |
| **High motivation (4–5)** | Still shrink — motivation will fade | Strong candidate — keystone | Strong candidate — keystone |

Rule of thumb: design for the lowest-motivation day. If the habit only works on high-motivation days, it's a wish, not a habit.

## Common Failure Modes Table

| Symptom | Root cause | Recovery move |
|---------|-----------|---------------|
| Skipped 2 days in a row | Streak fragility — no recovery plan | Do minimum-viable today; redesign tomorrow |
| Habit forgotten | Anchor wasn't reliable | Re-anchor to a more reliable cue |
| Habit done but felt hollow | No reward designed | Add an immediate sensory reward (sip, tick, sound) |
| Strong start, full collapse by week 4 | Too many habits at full intensity from week 1 | Follow the 8-week ramp; minimum-viable for weeks 1–2 |
| Habit competes with another | Same anchor, same time slot | Sequence them or move one to a different anchor |
| Habit dropped after illness | No recovery move pre-designed | The "recovery stack" — fire the cue only, for 2 days |
| Travel breaks everything | Stack was location-dependent | Build a portable keystone for travel weeks |
| Family event derails the week | No B-day version | Define a B-day version at 30% intensity |
| Stopped tracking | Tracker not visible | Move the tracker to a forced-passage location (fridge, mirror, kettle) |
| "I forgot the point" | Identity statement not reviewed | Re-read identity statement at next monthly review |
