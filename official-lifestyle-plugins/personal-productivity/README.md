# personal-productivity

4 lifestyle skills for designing daily and weekly systems that compound.

| Skill | Purpose |
|---|---|
| `habit-stacker` | Stack new habits onto existing routines with cue/reward design, accountability tricks, 8-week ramp |
| `sunday-reset` | Build a personalised weekly-reset ritual — reflection prompts, carryforward, week-ahead prep |
| `deep-focus-day` | Design a deep-focus day with distraction guardrails and energy-matched task batching |
| `energy-detective` | Map energy, focus, and mood across day & week from a 7-day log; surface drains and fixes |

Pure reasoning skills — no external APIs, no credentials required.
