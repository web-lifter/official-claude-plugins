# references — bmc-link-vpc

Connects two of the canonical Strategyzer canvases:

- **Osterwalder, Alexander & Pigneur, Yves.** *Business Model Generation.* Wiley, 2010. — the BMC's Customer Segments and Value Propositions cells.
- **Osterwalder, Alexander; Pigneur, Yves; Bernarda, Greg; Smith, Alan.** *Value Proposition Design.* Wiley, 2014. ISBN 978-1-118-96805-5. — the Value Proposition Canvas (customer profile: jobs / pains / gains; value map: products & services / pain relievers / gain creators).

The two canvases are designed to be used together: each BMC segment maps to one VPC, and each VPC value-map back-fills the BMC's Value Propositions cell.

See `startups/SOURCES.md` for the full bibliography.
