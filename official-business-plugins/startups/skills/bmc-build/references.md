# references — bmc-build

The Business Model Canvas is a one-page nine-cell template introduced in:

- **Osterwalder, Alexander & Pigneur, Yves.** *Business Model Generation.* Wiley, 2010. ISBN 978-0-470-87641-1.

The skill treats the canvas as a **hypothesis sheet** — every cell entry is tagged `hypothesis` or `fact`, never bare assertion. A `fact` tag requires an evidence link to a concluded learning card (see `venture-experimentation/learning-card-build`).

The nine cells (right side = front-stage / customer-facing; left side = back-stage / business-facing) are:

1. Customer Segments
2. Value Propositions
3. Channels
4. Customer Relationships
5. Revenue Streams
6. Key Resources
7. Key Activities
8. Key Partnerships
9. Cost Structure

See also `startups/SOURCES.md` for the full bibliography.

## Related skills

- `bmc-front-back-split` — renders the same canvas into the front-stage / back-stage view (also from *Business Model Generation*, Ch. 2).
- `bmc-update` — bumps the BMC to v(N+1) when hypotheses flip.
- `bmc-link-vpc` — bidirectional linkage with `value-proposition/vpc-build` (Osterwalder et al., *Value Proposition Design*, 2014).
