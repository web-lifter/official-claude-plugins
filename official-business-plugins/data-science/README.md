# experimentation

4 data-science skills for experiment design, readouts, forecasting, and causal inference.

| Skill | Purpose |
|---|---|
| `ab-test-designer` | Design A/B/n experiment — hypothesis, power, MDE, guardrails, decision criteria |
| `experiment-readout-builder` | Analyse results — significance, CI, segments, decision matrix |
| `forecasting-model-spec` | Time-series forecast spec — ARIMA / Prophet / ML — validation + monitoring |
| `causal-impact-analyser` | Quasi-experimental design — diff-in-diff, synthetic control, ITS, RDD |

Includes a `stats-reviewer` agent invoked for peer-review of p-hacking risks and design pitfalls.
