# strategic-economics

3 strategy-economics skills for product and venture decisions.

| Skill | Purpose |
|---|---|
| `competitive-dynamics-analyser` | Porter 5 Forces + game-theory primer; equilibrium prediction; response-game tree |
| `elasticity-estimator` | Price elasticity estimation method selection (historical / survey / experimental) |
| `moat-strength-audit` | Durability scoring across 7 moat types (network, switching, scale, brand, IP, data, regulatory) |

Includes `red-team-strategist` agent that challenges optimistic conclusions.
