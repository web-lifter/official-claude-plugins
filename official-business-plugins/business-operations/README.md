# business-operations

5 skills for SMB operations.

| Skill | Purpose |
|---|---|
| `revenue-channel-mapper` | Map every revenue channel onto a unified canvas with contribution %, CAC, LTV |
| `kpi-framework-generator` | North-Star → input metrics → functional KPIs tied to OKRs |
| `stakeholder-brief-builder` | Tight 1-page SCQA brief tailored to a stakeholder |
| `operational-bottleneck-detector` | Identify operational bottlenecks via Theory of Constraints + VSM |
| `pricing-strategy-analyser` | Recommend pricing model, tiers, and elasticity guard-rails |

Pure reasoning skills — no external APIs, no credentials required.
