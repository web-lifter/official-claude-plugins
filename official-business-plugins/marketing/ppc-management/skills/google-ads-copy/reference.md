# Google Ads Copy — Reference

---

## 1. Character limits (2026)

| Asset | Max chars | Count |
|---|---|---|
| Headline | 30 | 15 per RSA |
| Description | 90 | 4 per RSA |
| Path 1 / Path 2 | 15 each | 2 per RSA |
| Sitelink headline | 25 | 6 per campaign |
| Sitelink description 1 | 35 | |
| Sitelink description 2 | 35 | |
| Callout | 25 | 20 per campaign |
| Structured snippet header | fixed list | |
| Structured snippet value | 25 | 10 per snippet |

## 2. Pin positions

| Position | What goes here |
|---|---|
| HEADLINE_1 | Brand or primary keyword — always visible |
| HEADLINE_2 | USP — second-most-visible |
| HEADLINE_3 | CTA — third slot |
| DESCRIPTION_1 | Primary value proposition + CTA |
| DESCRIPTION_2 | Trust / proof |

## 3. Headline formulas

- `{Brand} {Product}` — "Koala Wool Throws"
- `{Product} for {Audience}` — "Wool Throws for Bedrooms"
- `{Product} — {USP}` — "Merino Throws — Made in AU"
- `{Action Verb} {Product}` — "Shop Handmade Throws"
- `{Benefit} from {Brand}` — "Warmth from Koala & Co."
- `{Price Point} {Product}` — "Throws From $129"
- `{Location-specific}` — "Aussie-Made Throws"

## 4. Description formulas

- `Benefit + CTA`: "Bring home winter warmth. Shop Koala throws with free shipping over $150."
- `Proof + CTA`: "Loved by 10,000 Australian homes. Explore the Koala throws collection."
- `Informational`: "Handmade wool and linen throws from Australian makers."
- `Urgency`: "Limited winter stock — new designs in 48 hours."

## 5. Copy anti-patterns

| Don't | Why |
|---|---|
| "The BEST throws in Australia!!!" | Caps + exclamation + superlative = Google disapproval |
| "Premium quality products" | Generic — could be anyone |
| "Click here" | Google rejects |
| "Free!!!" | Exclamation overload |
| "100% satisfaction guaranteed" | Needs literal proof |
| "Save up to 50%" without a specific offer | Could be disapproved |

## 7. Callout library

Pick 5 per campaign from:

- Free shipping over $X
- Fast Aussie delivery
- Made in Australia
- 30-day returns
- Handmade with care
- Small-batch production
- Sustainable materials
- Family-owned business
- Local customer support
- Easy returns
- Secure checkout
- 12-month warranty
- No minimum order
- Gift-wrapping available
