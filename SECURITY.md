# Security Policy

## Reporting a Vulnerability

If you discover a security issue in any plugin or in the marketplace tooling,
please email **john@weblifter.com.au** rather than opening a public issue. We
will acknowledge receipt within 72 hours.

Confirmed vulnerabilities are treated as P0: the affected plugin is delisted
from its marketplace catalogue (`official-business-plugins/.claude-plugin/marketplace.json`
or `official-lifestyle-plugins/.claude-plugin/marketplace.json`) pending
remediation.

## Handling of published content

Both marketplaces in this repository are fully public on GitHub. Do **not** add
anything private, secret, or customer-identifying to any plugin, skill, template,
or example — treat everything committed here as world-readable and permanent.
